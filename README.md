## 🗺️ What's Inside? (Website Sections)

Instead of reading through text here, feel free to visit the live link above to explore these interactive sections:

## 💻 How to Run Locally

If you want to view the source code environment on your own machine, follow these steps:

1. Clone this repository:
   ```bash
   git clone https://github.com/Suhani-01/React-Portfolio
   ```
2. Navigate into the folder:
   ```bash
   cd React-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
